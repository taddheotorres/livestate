# Livestate — Frontend

Angular 17 SPA para la plataforma Livestate.

## Desarrollo

```bash
npm install
npm start
```

## Producción

```bash
npm run build -- --configuration production
```

Los artefactos se generan en `dist/frontend/`.

## Entornos

| Entorno | URL |
|---|---|
| Producción | [livestate.up.railway.app](https://livestate.up.railway.app) |
| Local | `http://localhost:4200` |
